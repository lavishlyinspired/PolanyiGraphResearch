# Polanyi Works API Server

The FastAPI backend. The implementation is the `polanyi.api` package in this directory; `main.py` exposes it for `uvicorn apps.server.main:app`. `polanyi serve` runs the same app.

Code: `polanyi/api/` — part of the single `polanyi` distribution
(mapped in the root `pyproject.toml`).
